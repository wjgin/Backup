1. 강의 내용

	<String 정의 및 특징>

	- String s = new String("문자");
	- String s = "문자";
	
	- 문자열은 변수 값을 직접 저장하는 것이 아닌 변수 값의 위치(주소)의 참조 값을 저장한다.
		- s1 = "hello";
		- s2 = "hello"; 의 경우  s1, s2는 같은 참조 값을 가지고 있다.
		- "hello" 문자열은 메모리 어딘가에 하나만 저장 되어있음.
