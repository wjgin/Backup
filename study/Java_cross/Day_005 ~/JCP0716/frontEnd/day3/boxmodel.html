Flex 컨테이너에 적용하는 속성들

display: flex;
Flex 컨테이너에 display: flex;를 적용하는 속성
flex-direction
row: 아이템들을 가로 방향 배치
row-reverse : row 방향의 반대 방향 배치
column : 세로 방향 배치
column-reverse : 세로 방향 역순 배치
flex-wrap
컨테이너가 더 이상 공간이 없을 때, 아이템을 한줄 넘기는 속성
flex-wrap: nowrap; 넘기지 않음
flex-wrap: wrap; 아래로 넘김
flex-wrap: wrap-reverse; 위로 넘김
flex-flow
flex-direction과 flex-flow를 함께 지정 가능한 속성
flex-flow: row wrap;
flex-direction: row;
flex-wrap: wrap;
justify-content
메인축 방향을 정렬 하는 속성
justify-content: flex-start;
justify-content: flex-end;
justify-content: center;
justify-content: space-between;
justify-content: space-around;
justify-content: space-evenly;
align-items
align-items: stretch;
align-items: flex-start;
align-items: flex-end;
align-items: center;
align-items: baseline;
